# 🤖 ИИ-консультант профсоюзной организации ХГУ

**Интеллектуальная система для автоматической консультационной поддержки студентов Первичной профсоюзной организации обучающихся ХГУ на основе генеративных нейросетей (YandexGPT).**

---

## 📌 Назначение

Система предназначена для автоматизации ответов на типовые вопросы студентов, связанных с деятельностью профсоюза:

- Защита прав обучающихся
- Материальная и социальная поддержка
- Проживание в общежитиях
- Организация мероприятий
- Предоставление скидок от партнёров

ИИ-консультант разгружает штатных сотрудников профсоюза и позволяет студентам получать оперативные ответы в любое время через официальное сообщество ВКонтакте.

---

## 🧠 Как работает

1. **Пользователь** задаёт вопрос через интерфейс ВКонтакте.
2. **Система** определяет, является ли запрос командой (кнопки меню) или свободным вопросом.
3. **YandexGPT** (LLM) генерирует ответ на основе общей информации о профсоюзной деятельности.
4. **Кнопочное меню** позволяет запускать специализированные сценарии:
   - Калькулятор стипендии
   - Проверка членства в профсоюзе
   - Часто задаваемые вопросы
5. **Административная панель** позволяет управлять данными: членами профсоюза, настройками стипендий, мероприятиями.

---

## 🧩 Основные компоненты

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Бэкенд | Flask (Python) | Приём и обработка запросов от ВКонтакте |
| База данных | PostgreSQL | Хранение членов профсоюза, настроек стипендий, мероприятий |
| Генерация ответов | YandexGPT API | Формулирование ответа на свободные вопросы |
| Админ-панель | Flask-Admin + SQLAlchemy | Управление данными профсоюза |
| Интеграция с ВК | VK API | Приём и отправка сообщений |

---

## 🚀 Функциональные возможности

- ✅ Приём и обработка сообщений через VK API
- ✅ Генерация ответов на свободные вопросы через YandexGPT

- ✅ Проверка членства в профсоюзе
